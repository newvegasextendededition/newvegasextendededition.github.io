УСТАНОВКА FALLOUT EXTENDED EDITION

Спасибо за проявленный интерес! Данная страница поможет Вам с установкой.

ПЕРЕД НАЧАЛОМ

  • Достижения и Steam Cloud работают если есть купленная игра в Steam, MO2 все сам подключит, скачанная игра НЕ ТРЕБУЕТСЯ
  • Сборка автономна, лицензия или скачанная игра НЕ требуются
  • Все DLC и Pre-Order паки включены, спрятаны по миру, где именно смотрите в разделе ПОМОЩЬ
  • Новая игра обязательна, старые сохранения НЕ поддерживаются
  • Для обладателей европейской версии (без приставки PCR) в Steam - потребуется переключить на EU в настройках сборки
  • Функции EGS и GOG НЕ поддерживаются
  • Требуется Windows 10 64-bit и выше, Visual C++, DirectX, обновленные драйвера видеокарты; 20 ГБ для загрузки, 49 ГБ для установки
  • Начинать новую игру рекомендуется после каждого КРУПНОГО обновления сборки
  • Рекомендуемый торрент-клиент для скачивания сборки: qBittorrent

ЗАПУСК И НАСТРОЙКА УСТАНОВЩИКА

  • Скачиваем торрент (зеркало), запускаем "Fallout New Vegas Extended Edition.exe".

  • Выбираем путь установки сборки, жмем далее.

    ВАЖНО

    В пути установки не должно быть кириллицы, нельзя ставить в Program Files и Program Files (x86) и в любые другие системные папки.

  • Жмем установить и ждем окончания.

ИГРАЕМ

  • Запускаем "New Vegas EE" с рабочего стола.

  • Настраиваем сборку в разделе "Настройки сборки" по своему желанию и играем.
